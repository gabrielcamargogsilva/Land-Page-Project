# The Digital Atelier - Landing Page Responsiva

## 📋 Sobre o Projeto

Este é um projeto acadêmico desenvolvido durante a faculdade com o objetivo de aprimorar conceitos de **responsividade** e **design adaptativo**. A landing page "The Digital Atelier" demonstra como criar uma página que se adapta perfeitamente a diferentes tamanhos de tela, oferecendo experiências otimizadas para dispositivos móveis e desktops.

## 🎯 Objetivo da Atividade

Consolidar aprendizados sobre:
- ✅ Design responsivo com CSS Media Queries
- ✅ Adaptação de navegação para diferentes dispositivos (menu mobile vs desktop)
- ✅ Flexibilidade de layouts com Flexbox
- ✅ Otimização de componentes para diferentes resoluções
- ✅ Boas práticas de UX e UI responsiva


## 💻 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e responsividade
- **Google Fonts** - Fonte "Plus Jakarta Sans"
- **Media Queries** - Adaptação responsiva

## 🎨 Design Responsivo

### Versão Mobile
- Menu hambúrguer compacto no topo
- Navegação simplificada com ícones
- Layout em coluna
- Imagens e conteúdo otimizados para toque
- Botões em tamanho adequado

### Versão Desktop
- Navegação completa com links textuais
- Menu fixo no topo
- Layout multi-coluna
- Melhor aproveitamento do espaço
- Ícones adicionais (perfil)

## 📱 Pontos-Chave de Responsividade

1. **Menu Adaptativo**: Menu mobile oculto no desktop e vice-versa
2. **Flexbox Layout**: Arranjo flexível que se adapta ao espaço disponível
3. **Imagens Responsivas**: Uso de `max-width: 100%` para escalabilidade
4. **Tipografia Fluida**: Tamanhos ajustáveis conforme a viewport
5. **Espaçamento Dinâmico**: Padding e margin adaptados por breakpoints

## 🚀 Como Executar

1. Clone ou faça download do projeto
2. Abra o arquivo `index.html` em seu navegador
3. Teste a responsividade redimensionando a janela ou usando as ferramentas de desenvolvedor (F12)

## 📏 Breakpoints Recomendados

- **Mobile**: até 768px
- **Desktop**: acima de 769px

## 📚 Conceitos Aplicados

- Viewport meta tag para mobile
- Mobile-first approach
- CSS Grid e Flexbox
- Unidades relativas (rem, em, %)
- Box-sizing: border-box
- Hover states para desktop

## ✨ Resultado Final

A página oferece uma experiência fluida em qualquer dispositivo, com transições suaves entre layouts e mantendo a identidade visual em todos os tamanhos de tela.
